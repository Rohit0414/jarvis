Precision Filtering (Domains)

Export only what matters. If you don't add a filter, the module exports everything.

Syntax Examples
Only Confirmed Sales Orders:
[('state','=','sale')]
Specific Company + Threshold:
[('company_id','=',1),('amount_total','>',1000)]
Best Practices
  • Start without a filter to confirm the basic sync works.
  • Once confirmed, add filters to keep the Snowflake table lean.
  • Developer Hack: Use Odoo's built-in search filters in list view to verify your domain logic first.
üí° Filters lower your Snowflake storage and compute costs.
Step 3

Automation & Delta Sync

Smart Sync Logic

Incremental means we only talk about what changed. Odoo tracks timestamps so we don't waste bandwidth re-sending old records.

Interval Minutes

How often should the engine run? 15 minutes is the standard sweet spot. Increase to 30-60 minutes for multi-million row tables.

Automation Tips
  • Enable Auto-Export only for production tables.
  • You can still click Run Export manually at any time.

Re-Exporting Logic

Exporting again is safe. If a row changed in Odoo, the Snowflake row updates automatically. No manual deletions needed.

Fixing Problems

Most sync failures are configuration-based. Run through these checks if you encounter errors.

Validation Checklist
  • Is the Account ID string exactly ID.Region?
  • Are credentials (User/Pass) valid?
  • Does the Warehouse name exist exactly in Snowflake?
  • Has the Snowflake User been granted Schema permissions?
Common Warnings

Permission Error: Ask your Snowflake admin to allow: CREATE TABLE, UPDATE, and MERGE.

Frequently Asked Questions

Will it delete my data?

No. The module uses UPSERT logic to update old rows and safely add new ones.

Handle big data?

Yes. It processes data in batches and only sends incremental changes to maintain performance.

Custom models?

Absolutely. Any model visible in Odoo, including custom ones, can be exported.

Do I need SQL?

No. Everything is managed through the Odoo UI by picking models and columns.

Stop auto export?

Yes. Simply toggle the "Auto Export" switch to OFF in the Export profile.

Where are the tables?

Log into Snowflake and navigate to: Database ‚Üí Schema ‚Üí Tables.
