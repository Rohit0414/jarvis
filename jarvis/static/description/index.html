SMART DATA MAPPING

Intelligent Schema Handling

Our connector intelligently translates complex Odoo field types into clean, BigQuery-compatible columns, saving you countless hours of manual data wrangling and ensuring data integrity from source to insight.

  • Many2one Fields: Automatically exports both the related record's `ID` (e.g., `partner_id`) and its human-readable `Name` (e.g., `partner_name`).
  • Selection Fields: Converts internal Odoo keys (e.g., `'draft'`) to their user-friendly labels (e.g., `'Draft'`), making your data instantly understandable.
  • Date & Datetime: Ensures all date/datetime fields are formatted to ISO 8601 standards for seamless time-series analysis in your BI tools.
  • Numeric & Boolean: Automatically maps Odoo numeric types to BigQuery `INTEGER`, `BIGNUMERIC`, or `FLOAT`, and boolean fields to `BOOL`.

SEAMLESS DEPLOYMENT

Installation & Effortless Configuration

From initial installation to your first successful data sync, get up and running in just a few straightforward steps.

Quick Start Guide:
  1. Acquire the Module: Purchase and download from the Odoo App Store.
  2. Install in Odoo: Add to your addons directory and install via Odoo's Apps menu.
  3. Google Cloud Setup: Create a Service Account and grant it "BigQuery Data Editor" and "BigQuery Job User" roles.
  4. Secure Key Upload: Navigate to Settings → BigQuery Connector in Odoo and upload the JSON key for your Service Account.
  5. Configure & Run: Create your first export profile, select the model and columns, set filters, and run your first sync!
Compatibility & Requirements
  • Odoo Version: 14, 15, 16+
  • Google Cloud: Active GCP Account with BigQuery API enabled.
  • Connectivity: Internet access for API calls to Google Cloud.
  • Deployment: Works with On-premise & Cloud hosting.
  • Security: We recommend a dedicated Service Account.
FAQ

Frequently Asked Questions

Find quick answers to common questions about our connector.

What do I need to use this module?

You need an Odoo instance (v14+), a Google Cloud account with BigQuery enabled, and a Service Account from that project with "BigQuery Data Editor" and "BigQuery Job User" permissions.

How is my data kept secure?

Data security is paramount. The connector establishes a direct, encrypted connection (via HTTPS) between your Odoo server and Google's API. Your data is never routed through third-party servers, and access is controlled by the Google Cloud Service Account key that you provide.

Does it work with my custom modules and fields?

Absolutely. The connector dynamically reads your entire Odoo database schema, including any custom modules you have installed and custom fields you have added. You can select these for export just like standard Odoo fields.

What are the costs involved?

You purchase the Odoo to BigQuery Connector license one-time from the Odoo App Store. The only ongoing costs are for your Google BigQuery usage (storage and querying), which is billed directly by Google Cloud. There are no recurring subscription fees from us.

Can I schedule automatic exports?

Yes. The module leverages Odoo's built-in automation server (Scheduled Actions). You can configure any export profile to run on a schedule you define (e.g., hourly, daily, weekly) for fully automated, incremental data syncs.
