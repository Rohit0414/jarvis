Frequently Asked Questions

How does the Incremental Sync work?

The module stores the timestamp of the last successful export. In the next run, it only queries records where write_date > last_export_date. This keeps the load light and the sync fast.

Does it support Many2one fields?

Yes. Many2one fields are exported as their ID. You can also export the "name" field or other related data by defining them in the query builder.

What permissions are needed in Snowflake?

The Snowflake user requires USAGE on the Warehouse, Database, and Schema, and CREATE TABLE / INSERT / UPDATE / DELETE permissions on the target schema.

Does this support Odoo.sh?

Absolutely. It works on Odoo.sh, On-Premise, and Odoo Online (if external libraries are allowed via your hosting plan). You only need to install the snowflake-connector-python library.

Will it slow down my Odoo instance?

No. The sync runs as a background job (Cron). We recommend scheduling heavy data syncs during off-peak hours for massive initial loads.

Can I export to multiple Databases?

The current version connects to one Database/Schema defined in Settings. However, you can create multiple queries targeting different tables within that schema.
