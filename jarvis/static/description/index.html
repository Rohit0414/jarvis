SEAMLESS DEPLOYMENT

Installation & Effortless Configuration

Get up and running in just a few straightforward steps, designed for both technical and business users.

Quick Start Guide:
  1. Acquire Module: Purchase and download from the Odoo App Store.
  2. Install in Odoo: Add the module to your addons directory and install.
  3. Google Cloud Setup: Create a Service Account and grant it "BigQuery Data Editor" & "BigQuery Job User" roles.
  4. Secure Key Upload: In Odoo, navigate to Settings → BigQuery Connector and upload the JSON key.
  5. Configure Export: Create an export profile, select a model, choose columns, and initiate your first sync!
Requirements
  • Odoo Version: 14, 15, 16, and newer.
  • Google Cloud: Active account with BigQuery API enabled.
  • Connectivity: Internet access from your Odoo server.
  • Deployment: On-premise or Cloud hosting.
