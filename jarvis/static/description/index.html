Step 1: Setup Connection

Configuration
Where is the Settings page?

Go to: Settings ‚Üí Snowflake Connector Pro

Fill these fields:
  • ‚óè Account: Your Snowflake account ID string
  • ‚óè User: Snowflake username
  • ‚óè Password: Snowflake password
  • ‚óè Warehouse: e.g., COMPUTE_WH
  • ‚óè Database: e.g., ODOO_DB
  • ‚óè Schema: e.g., PUBLIC
  • ‚óè Role: Optional permissions role
How to find your Account (Step 1)

In Snowflake, open a worksheet and run this query:

SELECT CURRENT_ACCOUNT(), CURRENT_REGION(), CURRENT_ORGANIZATION_NAME();

Combine Account ID and Region (e.g., HN81816.ap-south-1) and enter this in the Odoo "Account" field.

Domain Filter (Optional)

A domain filter means: ‚Äúexport only some rows‚Äù. If you don‚Äôt add a filter, it exports everything.

Filter Examples
Confirmed Sales only:
[('state','=','sale')]
Company 1 & Amount > 1000:
[('company_id','=',1),('amount_total','>',1000)]
Optimization Tips
  • Start without a filter to confirm the connection works.
  • Add filters to keep Snowflake tables lean and minimize costs.
  • Use Odoo's built-in search filters to test your logic before adding here.
üí° Filters help keep Snowflake small and efficient.

Step 3: Auto Export (Incremental)

Automation
What is Incremental Sync?

This means exporting only what is new or changed. Old rows are not re-sent, saving bandwidth and Snowflake credits.

Interval Minutes

Defines how often Odoo checks for changes. 15 minutes is standard. Use 30-60 minutes for very large datasets.

Manual vs Auto

You can keep auto-export OFF and manually click Run Export whenever needed. Both options are safe and incremental.

Fix Problems

Most problems are related to wrong settings or missing permissions. Use this checklist to troubleshoot.

Technical Checklist:
  • Account ID correct? (Must include region)
  • Credentials verified?
  • Warehouse/DB/Schema names match exactly?
  • Does the Snowflake user have CREATE TABLE and MERGE permissions?
Error logs:

If you encounter an error, check the Snowflake logs in Odoo for the specific error message.

Frequently Asked Questions

Will it delete data?

No. It updates old rows and adds new rows safely.

Handle big data?

Yes. It sends data in batches and only sends incremental changes.

Custom models?

Yes. Both standard and custom models are fully supported.

Do I need SQL?

No. Simply choose the model and columns from the UI.

Stop auto export?

Yes. Turn off the Auto Export toggle in the profile.

Where are tables?

In Snowflake: navigate to Database ‚Üí Schema ‚Üí Tables.
